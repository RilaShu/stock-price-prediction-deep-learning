# skip-gram-Chinese
- **概要**<br>
针对中文语料数据，基于tensorflow的skip-gram算法实现,实验语料使用金庸全集（可替换）
- **代码**<br>
skipgram_chinese.py -- 源码<br>
usage_example.py -- 使用示例（需下载word2vec.txt）<br>
- **语料与模型**<br>
语料 -- 金庸全集<br>
模型 -- word2vec.txt (10万词，100维向量表示)<br>
文件较大，均提供外链下载<br>
- **效果示例**<br>

pd.Series(word2vec_model.most_similar(u'乔峰'))
